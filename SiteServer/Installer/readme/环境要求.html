环境要求


操作系统要求

    Windows Server 2008、Windows Server 2003、Windows 2000、Windows XP

语言及数据库支撑环境要求

    安装 SiteServer 系列产品需要装有如下软件：

    • IIS 5.0、IIS6.0、IIS7.0及以上版本
    • .NET Framework 2.0、.NET Framework 3.5、.NET Framework 4.0
    • SQLSERVER 2000 、SQLSERVER 2005 或 SQLSERVER 2008 及以上版本

    如果您租用虚拟主机，请咨询虚拟主机提供商，您的空间服务器是否已安装了上述软件。

    您的 SQL Server 数据库账号应当拥有"当前数据库管理员"权限，同时文件空间不低于 60M，数据库空间不低于 30M。

    由于 SiteServer 系列产品的数据表具有前缀设计，因此通常情况下可以将 SiteServer 系列产品与其他软件安装在同一个数据库中。

安装可能用到的工具软件

    如果您通过网络将 SiteServer 系列产品软件上传到服务器上，您将可能需要一个 FTP 客户端软件。通过您的服务器 FTP 账号，使用该 FTP 客户端软件将相关文件上传到服务器上。
