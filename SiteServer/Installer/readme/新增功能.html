新增功能


SiteServer 系列产品 3.5 版本修正和改进的问题包括：

  • CMS站点配置里面新增给用户发送站内信功能
  • 后台内容(栏目)管理界面，新增显示"所属内容（栏目）组"选项
  • stl:navigation新增isKeyboard属性，实现键盘左右键实现内容上一页下一页自动翻页
  • 后台点击"内容组"或"栏目组"，会显示出相应的"内容"或"栏目"
  • 设置内容（栏目）组界面，新增新建内容（栏目）组功能
  • stl:file新增isFilesize属性，计算附件大小
  • "内容管理"，新添"整理"，可以按照"内容ID"或"添加日期"给内容重新排序
  • Access版本内容点击量一直是0
  • Access版本"信息管理"----"内容搜索"，按时间搜索报错
  • Access版本，"用户中心"----"我的投稿"报错
  • Access版本，"站点配置"---->"站点用户管理"---->"用户管理"，注册时间和最后活动时间报错
  • Access版本，"成员权限"---->"管理员管理"---->"管理员管理"，创建日期和最后登录日期报错
  • Access版本，"功能管理"---->"流量统计管理"---->"站点流量统计"---->"栏目流量统计"报错/li>
  • Access版本，"信息管理"---->"站点数据统计"---->"管理员工作量统计"报错
  • Access版本，"信息管理"---->"站点数据统计"---->"用户投稿量统计"报错
  • "信息管理"---->"站点数据统计"---->"内容点击量排名"，解决脚本错误
  • Oracle版本，内容附件名称乱码
  • GB2312下，用Request时接收不到中文参数
  • 栏目虚拟字段的值添加不进去
  • 在新建字段的时候，会自动添加value值

  • 用户投稿FCK编辑器上传的flash无法添加到内容里
  • stl:pageinputcontents什么都不显示，调不到提交表单里的内容

  • stl:login火狐退不出去
  • "栏目字段管理"无法删除虑拟字段
  • 绝对路径下GB2312编码下搜索不到内容
  • 提交表单无法上传附件
  • 生成管理-生成栏目页-删除全部栏目页面-删除全部内容页面，将会把分页的及空文件夹也删除掉
