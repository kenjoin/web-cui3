安装SiteServer 系列产品


第 1 步：上传或解压程序包

    使用 FTP 软件登录您的服务器，建立一个单独的目录，或选择合适的位置，确保存放在此位置的文件能够被 web 请求所访问到，并且该目录中具有执行 aspx 代码的权限。
    将 SiteServer 系列产品文件包的 ./upload_files 目录中的全部文件和目录结构上传到服务器（注意是上传 upload_files 目录中的文件和目录，而不是上传包含 upload_files 目录本身的目录和结构）；或者使用Winrar、Winzip等软件解压程序包并存放到您的磁盘目录上。

第 2 步：设置目录权限

    安装 SiteServer 系列产品时请使用NTFS磁盘格式，否则在安装中可能出现异常。

    Windows Server 2003:请您在安装产品前给网站根目录添加IIS来宾帐号（IUSR_Hostname）的只读权限，以及Network Service组的读写修改权限。

    Windows Server 2000:请您在安装论坛前给网站根目录添加IIS来宾帐号（IUSR_Hostname）的只读权限，以及ASP.NET帐号的读写修改权限。

    Windows XP:请您在安装论坛前给网站根目录添加Network Service组的读写修改权限。

    详细权限设置请参考《目录权限设置(简单)》或《目录权限设置(高级)》。

    设置完毕目录权限后，我们需要在Windows控制面板 -> 管理工具 -> Internet 信息服务(IIS)管理器中进行IIS站点的建立及配置。详细说明可见文档中心的《建立IIS站点》。

    如果SiteServer 系列产品是SQLSERVER 版本，需要创建数据库，详细说明可见文档中心的《建立 SQL SERVER 数据库》。

    虚拟主机的用户请联系虚拟主机商进行目录权限设置更改。

第 3 步：访问安装向导进行安装

    请在浏览器中运行 siteserver/installer/index.aspx，即访问 http://您的域名/siteserver/installer/index.aspx ，然后按照提示操作完成最后的安装。

    详细安装步骤请参考《安装 SiteServer 系列产品》

第 4 步：创建网站

    系统安装完毕后将提示您创建第一个网站，如果选择站点模板将创建与所选模板完全相同的站点。

    至此，安装完毕。
