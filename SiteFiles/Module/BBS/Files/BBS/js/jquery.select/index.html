简介 1、根据原有的select 模拟生成新的 select。
2、支持键盘，可通过键盘选择选项。
3、可自定义样式。
4、方便灵活，可选择需要的seletc 进行模拟（非全局模拟）
5、判断用户是否开启脚本支持，若未开启，不进行模拟。
6、测试通过IE6+,Firefox 3.5.5,Chrome 3.0.1,Opera 10.0,Safari 4.0.4
